官方解决方案



For Windows Vista users:  

If the Flash uninstall / reinstall does not work this maybe due to a Microsoft

Windows Vista permissions issue.  

  

Please use the Windows button at the bottom left then the search function:  

  

-Go to: C:\Windows\System32\Macromed\Flash\\.   

-Right-click on the following file: FlashUtil10b.exe or FlashUtil10e.exe.   

-Select "Run as Administrator" and install the update..   

-Restart your computer.  

大意：如果你是windows Vista（或win7）用户，打开 C:\Windows\System32\Macromed\Flash\\.

，找到FlashUtil10b.exe or FlashUtil10e.exe文件，右键打开，以管理员身份运行，好了重启电脑就行了
